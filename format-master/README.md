# 格式大师 FormatMaster

智能文档格式修改器 - 一个帮助你快速检测和修复文档格式问题的Web应用。

## 功能特点

- 📁 **文件上传**: 支持拖拽或点击上传 PDF、Word(.docx)、纯文本(.txt) 文件
- 📋 **模板模式**: 内置6种常用文档模板，每个模板都有详细的格式规范说明
  - 学术论文格式（GB/T 7714）
  - 国家标准公文格式（GB/T 9704）
  - 政府报告格式
  - 海报排版（竖版）
  - 海报排版（横版）
  - 商务报告格式
- ✨ **自定义需求**: 用自然语言描述你的格式要求，系统智能解析关键词
- 🔍 **格式分析**: 自动检测文档中的格式问题
- 🔧 **一键修复**: 快速修复所有检测到的格式问题
- 📥 **报告导出**: 下载格式分析报告或修复报告

## 技术栈

- HTML5 + CSS3 + JavaScript (原生)
- PDF.js - PDF文件解析
- Tailwind CSS 风格的样式设计

## 快速开始

### 方法一：直接在浏览器中打开

1. 进入 `format-master` 文件夹
2. 双击 `index.html` 文件
3. 在浏览器中打开即可使用

### 方法二：使用本地服务器

如果你想要完整的 Word 文档支持（需要 mammoth.js）：

```bash
# 使用 Python 启动简单服务器
cd format-master
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后在浏览器中访问 http://localhost:8080
```

## 使用说明

### 1. 上传文件

将文档拖拽到上传区域，或点击"选择文件"按钮。

### 2. 选择模式

- **模板模式**: 从预设的6种模板中选择，每个模板都有详细的格式规范展示
- **自定义需求**: 用自然语言输入格式要求，如"字体用微软雅黑，字号14磅，行距1.5倍"

### 3. 开始分析

点击"开始分析"按钮，系统会自动检测文档中的格式问题。

### 4. 修复并下载

查看分析结果，点击"修复全部问题"或单独修复每个问题，然后下载修复后的报告。

## 注意事项

1. **PDF文件**: 使用 PDF.js 解析，可以提取文本内容进行格式检测
2. **Word文件**: 当前版本为演示版，Word解析功能需要额外配置 mammoth.js
3. **文本文件**: 完全支持，可进行完整的格式分析

## 自定义需求示例

```
字体用微软雅黑，字号14磅，行距1.5倍，首行缩进2字符，标题居中加粗
```

系统会自动识别：
- ✅ 字体 → 微软雅黑
- ✅ 字号 → 14磅
- ✅ 行距 → 1.5倍
- ✅ 缩进 → 2字符
- ✅ 标题 → 居中 + 加粗
