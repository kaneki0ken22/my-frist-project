# JavaScript 零基础学习指南

专为 Web 开发零基础学生设计的 JavaScript 学习路径。

## 学习前准备

在开始 JavaScript 之前，学生需要了解：

1. **HTML 是什么** - 网页的骨架
2. **CSS 是什么** - 网页的样式
3. **JavaScript 是什么** - 网页的交互逻辑

每个项目模板中都包含完整的 HTML 结构，学生只需关注 `<script>` 标签内的 JavaScript 代码。

## 学习路径（共 4 个阶段）

### 阶段 1：基础概念（第 1-2 周）
**目标**：理解变量、数据类型、基本运算

**必读**：[01-basics.md](references/01-basics.md)

**练习**：
- 变量声明与赋值
- 数字和字符串操作
- 简单的数学计算

**检验标准**：能写出计算器的基础运算逻辑

---

### 阶段 2：逻辑控制（第 3-4 周）
**目标**：掌握条件判断和循环

**必读**：[02-logic-control.md](references/02-logic-control.md)

**练习**：
- if/else 条件判断
- for 循环遍历
- while 循环

**检验标准**：能完成 Todo List 的添加/删除功能

---

### 阶段 3：函数入门（第 5-6 周）
**目标**：学会封装代码，理解函数的作用

**必读**：[03-functions.md](references/03-functions.md)

**练习**：
- 函数声明和调用
- 参数和返回值
- 箭头函数基础

**检验标准**：能将重复代码提取为函数

---

### 阶段 4：DOM 操作（第 7-8 周）
**目标**：让网页"动起来"

**必读**：[04-dom-basics.md](references/04-dom-basics.md)

**练习**：
- 获取页面元素
- 修改元素内容
- 事件监听（点击、输入等）

**检验标准**：能完成交互式网页项目

## 学习方法建议

### 1. 边学边做
不要只看不练。每学一个知识点，立即在浏览器控制台尝试。

### 2. 善用浏览器开发者工具
- 按 `F12` 打开开发者工具
- 在 Console（控制台）标签页直接输入 JavaScript 代码
- 可以看到即时的执行结果

### 3. 遇到问题怎么办
1. 检查拼写（JavaScript 区分大小写）
2. 检查括号是否成对
3. 使用 `console.log()` 打印中间结果
4. 逐行阅读错误信息

### 4. 学习节奏
- **每天学习 1-2 小时** 比周末突击 8 小时更有效
- **完成小练习** 比看完长教程更重要
- **动手改代码** 比被动复制更有收获

## 代码示例使用指南

所有代码示例都可以：
1. **在控制台直接运行** - 打开 F12 → Console → 粘贴代码 → 回车
2. **在 HTML 文件中运行** - 使用 [html-template](assets/html-template/) 中的模板
3. **修改后观察结果** - 改变数值或逻辑，看看会发生什么
